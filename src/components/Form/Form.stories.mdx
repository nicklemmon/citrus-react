# Form

## Used for wrapping form elements

Use the `<Form/>` component to wrap form elements and handle form submission.

```jsx
import { Form } from '@citrus-react'

const MyComponent = () => {
  const submitData = () => {
    console.log('submit form!')
  }

  return (
    <Form onSubmit={submitData}>
      <input type="submit" />
    </Form>
  )
}
```
