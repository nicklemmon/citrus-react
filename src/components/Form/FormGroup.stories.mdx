# FormGroup

## Used for grouping smaller form components in to a functional whole

Use `<FormGroup/>` to wrap smaller form components for things like error handling and descriptions.

```jsx
import { FormGroup, FormLabel, FormField } from '@citrus-react'

const MyComponent = () => (
  <FormGroup id="first-name" variant="text" description="This is a description">
    <FormLabel>First Name</FormLabel>

    <FormField />
  </FormGroup>
)
```
