# Button

## Used for Attention-Grabbing Interactions

Use buttons for attracting the user's attention.

```jsx
import { Button } from '@citrus-react'

const MyComponent = () => <Button>Click Me</Button>
```

<Props of={Button} />

### Variants

#### Primary Variant

Used as the primary action on a page or within a form.
